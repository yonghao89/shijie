
###               视界开源监测管理系统 

视界开源: 为开源事业做贡献,拥抱开源的时代吧！

#### 主要特点:
- MVC架构，设计精巧，使用简单
- 使用mybatisplus封装
- 后台实现restful
- 使用 shiro 权限管理

#### 主要技术:
- 后台所用框架:SSM
- 前台所用框架: Jquery LayUI
- 弹出层layer.js
